# Road Accident Safety Intelligence System

An advanced local machine learning system for predicting and preventing road accidents in India. This project combines accident prediction with comprehensive safety analysis, route optimization, and emergency response planning.

## 🌟 Features

### 1. **Accident Risk Prediction** (Enhanced with AI Explanations)
- Predict accident risk levels (Low/Medium/High) based on conditions
- **SHAP-based explanations** showing which factors contribute most to risk
- **Safety scoring (0-100)** for any location and conditions
- Real-time risk assessment with confidence levels

### 2. **Route Safety Analysis**
- Compare **multiple route alternatives** between two locations
- Calculate **safety scores for entire routes** based on historical data
- Visual route comparison with color-coded safety indicators
- Recommends the **safest route** even if it's not the shortest

### 3. **AI-Driven Safety Recommendations**
- Infrastructure improvement suggestions (speed breakers, lighting, traffic signals)
- Traffic management optimization (signal timing, lane management)
- Location-specific action recommendations prioritized by impact
- Pattern detection from historical accident data

### 4. **Emergency Response Planning**
- Find **nearest hospitals** using OpenStreetMap data
- Calculate **ambulance ETA** considering traffic and road conditions
- Estimate response times and survival probability
- Rank hospitals by fastest response time

### 5. **Real-time Safety Scoring**
- Dynamic safety scores (0-100) adjusted for:
  - Historical accident patterns
  - Time of day and day of week
  - Weather conditions
  - Road type and speed limits

### 6. **Explainable AI**
- Transparent predictions with feature importance rankings
- Natural language explanations for risk assessments
- Visual breakdowns of contributing factors

### 7. **Geospatial Intelligence**
- OpenStreetMap integration for routing and location data
- Hospital and emergency facility mapping
- No paid API keys required (uses free OSM services)

## 📁 Files Description

### Core Application Files
- **`streamlit_app.py`**: Multi-tab web application (4 tabs: Risk Prediction, Route Analysis, Safety Recommendations, Emergency Response)
- **`train_notebook.py`**: Jupytext-style Python script to train the ML model
- **`model_utils.py`**: Feature engineering transformers and preprocessing utilities

### Advanced Feature Modules
- **`geospatial_utils.py`**: OpenStreetMap integration (geocoding, hospital search, routing)
- **`safety_scorer.py`**: Safety scoring algorithms and risk assessment
- **`explanation_engine.py`**: SHAP-based  model interpretability and explanations
- **`route_engine.py`**: Multi-route analysis and safety comparison
- **`ambulance_predictor.py`**: Emergency response time estimation
- **`ai_recommendations.py`**: Pattern detection and safety improvement suggestions

### Data & Configuration
- **`accident_prediction_india.csv`**: Historical accident dataset
- **`model.joblib`**: Trained model pipeline (generated by training script)
- **`location_counts.json`**: Accident frequency by location
- **`requirements.txt`**: Python dependencies
- **`example_input.json`**: Sample input format

### Documentation
- **`README.md`**: This file
- **`guide.md`**: Step-by-step setup and usage instructions
- **`technical.md`**: Technical documentation on algorithms and models
- **`project_structure.md`**: File organization tree

## 🚀 Quick Start

1. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```

2. **Train the model** (if `model.joblib` doesn't exist):
   ```bash
   python train_notebook.py
   ```

3. **Run the application**:
   ```bash
   streamlit run streamlit_app.py
   ```

4. **Access in browser**: Open `http://localhost:8501`

## 💡 Technology Stack

- **Machine Learning**: scikit-learn, XGBoost, LightGBM, CatBoost
- **Explainability**: SHAP (SHapley Additive exPlanations)
- **Geospatial**: OpenStreetMap (Nominatim, Overpass API, OSRM)
- **Frontend**: Streamlit with Folium for interactive maps
- **Data Processing**: pandas, numpy

## 🔒 Privacy & Local Processing

- **100% local processing**: All ML predictions run on your machine
- **Free APIs only**: Uses OpenStreetMap's free services (no API keys needed)
- **No data collection**: Your queries stay on your computer
- **Works offline**: Except for OSM routing queries (which can be cached)
